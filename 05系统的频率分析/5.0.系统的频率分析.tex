\chapter{系统的频率分析}

上一章介绍了傅里叶变换，也介绍了信号的傅里叶变换和信号的频域分析。

本章用傅里叶变换分析系统。
首先分析频域角度上的系统响应，即“系统的频率响应”，给出“频率响应函数”概念。
再以RC电路为例，在频域角度分析系统特性。
最后用频率响应函数定义理想滤波器。

本章要点：
\begin{itemize}
    \item 系统的频率响应。
    \item 频率响应函数。
    \item 理想滤波器。
    \item Nuquist采样定理
\end{itemize}
